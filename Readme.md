🚀 Project Overview
This is a simple To-Do List web application built using PHP, MySQL, HTML, and CSS.
It allows users to add and view tasks, demonstrating basic backend development and database integration.
This project is designed for beginners to understand how frontend and backend work together in a real web application.

✨ Features


➕ Add new tasks


📋 Display all saved tasks


💾 Store tasks in MySQL database


🔄 Dynamic data rendering using PHP



🛠️ Technologies Used


🌐 HTML – Structure of the web page


🎨 CSS – Basic styling


⚙️ PHP – Backend logic


🗄️ MySQL – Database management



🔄 CRUD Workflow (Simple Explanation)
This project follows basic CRUD operations:


C (Create) ➝ User adds a new task using a form


R (Read) ➝ PHP fetches and displays all tasks from the database


(Update & Delete can be added in future improvements)


💡 Workflow:


User enters a task in the input field


PHP receives the data using POST method


Task is inserted into the MySQL database


PHP retrieves all tasks using a SELECT query


Tasks are displayed dynamically on the webpage
